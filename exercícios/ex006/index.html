Itálico / Ênfase

eu quero este termo em itálico (forma)

Nesta frase, temos um termo em ênfase usando a tag EM (semantica)

Caso eu digite uma frase em italico e ela ficar da mesma cor que as outras, então significa que a tag <em> está com um <style> dentro da tag <head>, que fará com que todas as tags <em> fiquem da mesma cor

Agora caso queira uma tag de uma cor específica, coloque <style> dentro da tag inicial, e escolha a cor apenas nessa tag

Texto inserido

Podemos usar a tag <ins> para indicar e sublinhar algumas palavras (semantica)

Texto deletado

Podemos usar a tag <del> para deletar palavrasl(semantica)

Texto sobrescrito

Para inserir coisas do tipo x20+3, a tag é <sup>(em cima)

Texto subscrito

Para inserir coisas do tipo H2O, a tag é <sub>(embaixo)

Style

Caso digitar uma frase e ela ficar da mesma cor que as outras da mesma tag, então significa que a tag em que ela se encontra está com um <style> dentro da tag <head>, que fará com que todas as tags que estejam la fiquem da mesma cor em toda a pagina

Neste parágrafo estou utilizando a tag <style>para deixar a frase toda colorida

Já nesse parágrafo estou utilizando apenas no itálico.

Marcações

A tag <mark> marca as palavras como um grife

É possível mudar a cor da marcação utilizando <style>
