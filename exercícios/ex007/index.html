Código-fonte

Para deixar as letras com igual espaçamento,utiliza-se a tag <code>

Para colocarmos aspas podemos utilizar a tag <q>

Este é um exemplo de blockquote, uma sitação mais longa do que a tag <q> e que também da um espaçamento no início do parágrafo. Também é possível citar a fonte de algo utilizando a tag <cite> dentro da tag <blockquote>
